# Search Intelligence Engine

## Mission

Identify organic-search opportunities by connecting business relevance, user intent, existing pages, topic coverage, search evidence, and conversion potential.

## Epics

| Epic ID | Epic | Outcome |
|---|---|---|
| SEARCH-001 | Search Evidence Provider Contract | Normalize Search Console, keyword, SERP, and demo provider data |
| SEARCH-002 | Query Normalization | Normalize terms, locations, modifiers, and duplicate query variants |
| SEARCH-003 | Intent Classification | Classify informational, commercial, transactional, navigational, local, and comparison intent |
| SEARCH-004 | Topic Clustering | Group query sets into topics users can understand and edit |
| SEARCH-005 | Page-Query Mapping | Connect existing pages to query/topic opportunities |
| SEARCH-006 | Content Gap Detection | Detect high-value service, audience, location, and comparison gaps |
| SEARCH-007 | Cannibalization Signals | Identify pages competing for similar intent |
| SEARCH-008 | Local Search Opportunities | Detect location/service page and local proof opportunities |
| SEARCH-009 | Commercial Opportunity Scoring | Score demand, relevance, competition, conversion potential, and confidence |
| SEARCH-010 | Search Opportunity UI | Show opportunity evidence without pretending estimates are exact |
| SEARCH-011 | Search-to-Recommendation Bridge | Convert opportunities into recommendation candidates |
| SEARCH-012 | Search Opportunity Brief Seed | Generate initial brief inputs from search opportunity evidence |

## First Vertical Slice

Start with the smallest slice that creates evidence, produces a decision, and gives the user a meaningful next action.
