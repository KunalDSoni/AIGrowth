# Search Intelligence Engine Templates

## Epic Acceptance Additions

- Output is normalized.
- Evidence is inspectable.
- Simulated and estimated data is labelled.
- User impact is clear.
- Tests cover the core decision logic.

## Story Pattern

As a user, I can understand `<engine-specific insight>`, so I know what action to take next.

## Task Pattern

- Inspect existing code.
- Reuse working implementation.
- Add model or adapter.
- Add evidence.
- Add scoring or conclusion.
- Add tests.
