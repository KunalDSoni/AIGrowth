# Technical SEO Engine

## Mission

Convert crawl evidence into deterministic, explainable SEO observations that identify barriers to discovery, crawling, indexing, usability, and machine understanding.

## Epics

| Epic ID | Epic | Outcome |
|---|---|---|
| TSEO-001 | Audit Rule Contract | Standardize rule inputs, outputs, versioning, severity, and evidence |
| TSEO-002 | Crawlability Checks | Broken links, blocked pages, redirect chains, redirect loops, orphan signals |
| TSEO-003 | Indexability Checks | Noindex, canonical mismatch, invalid canonical, robots conflicts |
| TSEO-004 | Metadata Checks | Missing, duplicate, long, short, weak, or conflicting title/description |
| TSEO-005 | Heading and Content Structure Checks | Missing H1, multiple H1s, thin content, heading-order signals |
| TSEO-006 | Image SEO Checks | Missing alt text, broken images, oversized-image signals |
| TSEO-007 | Structured Data Checks | Missing opportunities, malformed JSON-LD, conflicting entities |
| TSEO-008 | Internal Link Health | Broken internal links, poor anchors, weak service-to-topic links |
| TSEO-009 | Mobile and Accessibility Basics | Viewport, language, form labels, unclear link text |
| TSEO-010 | Duplicate Content and Template Detection | Duplicate metadata, near-duplicate pages, boilerplate-heavy pages |
| TSEO-011 | Issue Grouping and Deduplication | Consolidate noisy issues into user-friendly groups |
| TSEO-012 | Technical SEO Recommendation Inputs | Convert issues into recommendation candidates |

## First Vertical Slice

Start with the smallest slice that creates evidence, produces a decision, and gives the user a meaningful next action.
