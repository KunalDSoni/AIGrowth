# Competitor Intelligence Engine

## Mission

Classify competitors correctly and explain where they outperform or underperform using evidence, not vanity comparison tables.

## Epics

| Epic ID | Epic | Outcome |
|---|---|---|
| COMP-001 | Competitor Classification | Distinguish business, organic, local, AI-answer, and user-defined competitors |
| COMP-002 | Competitor Profile Model | Store competitor services, geography, positioning, website, and source |
| COMP-003 | Service Coverage Comparison | Compare service and audience page coverage |
| COMP-004 | Topic Coverage Comparison | Compare topic clusters, content depth, and freshness |
| COMP-005 | Technical Health Comparison | Compare crawlability, metadata, schema, and internal linking signals |
| COMP-006 | Trust Signal Comparison | Compare proof, case studies, reviews, credentials, and authority indicators |
| COMP-007 | AI Mention Comparison | Compare mention frequency and prominence across AI observations |
| COMP-008 | Citation Comparison | Compare first-party and third-party citation presence |
| COMP-009 | Competitor Gap Conclusions | Generate concise conclusions and strategic responses |
| COMP-010 | Competitor Evidence UI | Show supporting evidence behind each comparison |
| COMP-011 | User Correction Workflow | Let users correct competitor type and relevance |
| COMP-012 | Competitor-to-Recommendation Bridge | Convert competitor gaps into action candidates |

## First Vertical Slice

Start with the smallest slice that creates evidence, produces a decision, and gives the user a meaningful next action.
