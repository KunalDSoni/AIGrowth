# Citation Intelligence Engine

## Mission

Explain which sources AI answers cite, where the user's brand is absent, which competitors are cited, and what source/content improvements are defensible.

## Epics

| Epic ID | Epic | Outcome |
|---|---|---|
| CITE-001 | Citation Normalization | Normalize URLs, domains, canonical sources, and duplicates |
| CITE-002 | Source Classification | Classify first-party, competitor, marketplace, directory, media, review, forum, and unknown sources |
| CITE-003 | Citation Aggregation | Aggregate cited domains and pages across prompt families and platforms |
| CITE-004 | Citation Stability Metrics | Track repeated citations and source volatility |
| CITE-005 | First-Party Citation Gap | Detect prompts where competitors are cited but the user is absent |
| CITE-006 | Third-Party Source Gap | Identify useful non-owned sources where brand presence may matter |
| CITE-007 | Source Quality Signals | Label source reliability, freshness, relevance, and risk |
| CITE-008 | Citation-to-Content Gap Mapping | Connect missing citations to missing or weak first-party content |
| CITE-009 | Citation Evidence UI | Show cited answer, page, domain, prompt, platform, and timestamp |
| CITE-010 | Citation Recommendation Bridge | Convert citation gaps into ethical, evidence-backed actions |

## First Vertical Slice

Start with the smallest slice that creates evidence, produces a decision, and gives the user a meaningful next action.
