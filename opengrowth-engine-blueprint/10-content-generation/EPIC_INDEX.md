# Content Generation Engine

## Mission

Generate useful, editable, evidence-grounded marketing assets from approved briefs and recommendations while preserving user control and factual integrity.

## Epics

| Epic ID | Epic | Outcome |
|---|---|---|
| GEN-001 | Brief Contract | Standardize brief inputs from search, content, citation, and recommendation engines |
| GEN-002 | AI Provider Abstraction | Support mock provider and future OpenAI, Anthropic, Gemini, and local providers |
| GEN-003 | Metadata Generation | Generate title, description, H1, and rationale |
| GEN-004 | Service Page Generation | Generate service pages from business, search, and proof evidence |
| GEN-005 | Article Brief and Draft Generation | Generate structured briefs and drafts with claims marked |
| GEN-006 | FAQ and Schema Proposal | Generate truthful FAQ and JSON-LD proposals with validation |
| GEN-007 | Social and Email Repurposing | Repurpose approved content into LinkedIn, GBP, email, and ad copy |
| GEN-008 | Claim Verification Workflow | Flag unsupported claims, fabricated risk, and facts requiring user confirmation |
| GEN-009 | Versioning and Diff View | Compare original, generated, edited, approved, and rejected versions |
| GEN-010 | Human Review and Approval | Require review before export or publishing handoff |
| GEN-011 | Brand Voice Enforcement | Apply tone, vocabulary, exclusions, and compliance notes |
| GEN-012 | Generation Telemetry and Cost Controls | Track generation events, failures, limits, and provider cost metadata |

## First Vertical Slice

Start with the smallest slice that creates evidence, produces a decision, and gives the user a meaningful next action.
